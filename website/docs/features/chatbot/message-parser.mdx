# `MessageParser.cs`

Tokenizes and validates raw user input before it reaches the
[Controller](./controller). The tab you picked there stays selected here.

<Tabs groupId="codedoc">

<TabItem value="api" label="API Documentation" default>

```csharp
public static class MessageParser
{
    public static Token[] Tokenize(string raw);
    public static bool TryValidate(string raw, out string error);
}
```

</TabItem>

<TabItem value="howto" label="How To Use">

```csharp
if (MessageParser.TryValidate(input, out var error))
    var tokens = MessageParser.Tokenize(input);
else
    Console.WriteLine(error);
```

</TabItem>
