# 3D Showcase (Test Sandbox)

These are **throwaway test folders** that demonstrate the 3D model viewer with
several models spread across nested categories — exactly the structure you can
copy for real features. Delete the whole `website/docs/showcase/` folder (and the
matching `.gltf` files in `website/static/models/`) when you no longer need it.

## What's inside

- **Geometry Lab** — a sphere and a torus, plus an all-in-one gallery page that
  loads multiple models at once.
- **Mechanical Parts** — a cylinder and a cone.

Every model here is a small procedurally-generated **glTF** file in
`website/static/models/`. They were created by `.ignored/gen-models.js`.

:::tip Folder = sidebar category
The sidebar you see on the left is auto-generated from this folder tree. A folder
becomes a category (its label/order comes from `_category_.json`); each `.mdx`
file becomes a page. To make a model page editable in the browser CMS, add a
matching collection in `static/admin/config.yml`.
:::
